RPG Rhythm Game

NOTE! Hobby project / Learning project In this project i wanted to explore the idea of combining a rhythm game with RPG mechanics. Syncing attacks, animations and effects to the beat seemed like a fun challenge to me. I built a simple prototype in Unity where the player can attack enemies by pressing buttons in time with the music. This project is still very much a work in progress so the visuals will look pretty basic for now. I also wanted to explore making custom tools inside of Unity so I built a custom track editor to be able to create the beat tracks for the game.

Development GIFs

Mechanics & Flow

This shows the basic rhythm mechanics in action. The player must queue a attack after building up the beat meter in the bottom middle part of the screen. I have some debug yellow dots showing in this clip representing when im pressing the input in comparison to the beat.

This shows the custom track editor. The tool allows me to set up beat inputs and timings for each song. Everything you see here is built from scratch by me using Unity's custom toolset. The tool kept on expanding as I built it and now it has a lot of functionality.

The song is able to be played inside the tool and you can toggle metronome on or off. The track is also divided between buildup phases and defense phases. (When enemies do their attacks)

I liked the idea of being able to paint the notes in the track editor so I built a paint mode that allows me to click and drag to paint in notes. This made it much faster to create tracks.

Here you can see the painting section and the wave editor at the top moving in sync. I also the option to paint notes "live" while the song is playing which was really nice to be able to test out how the notes feel in real time and if they match the notes heard in the song.

Final Thoughts

This project provided a fun challenge of syncing gameplay mechanics to music and building custom tools inside the editor to make working on these parts easier. This was my first try on creating custom tools inside Unity and I was postively surprised by how flexible and extensive the options for creating custom tools inside Unity were.
